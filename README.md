# MyLarpManager

## Requirements:

### Backend :
* Maven
* Javac (11+)

### Frontend :
* NodeJS (maven installs it in the project folder)

Build and run the whole app in the "Warden-dev" directory:

`mvn spring-boot:run`

Build and run only the FrontEnd:

In the directory "Warden-dev\src\main\js" :

`npm run start`
